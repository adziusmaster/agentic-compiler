## Max in array

Write `arr_max(xs: (Array Num), size: Num) -> Num` returning the
largest element of the first `size` values of `xs`.

- Assume `size ≥ 1`.
- Works for any signed integers or real numbers.
- `arr_max([3,1,4,1,5,9,2,6], 8) = 9`.
- `arr_max([-7,-2,-5], 3) = -2`.
- `arr_max([42], 1) = 42`.

In Python, use the signature `arr_max(xs: list, size: int) -> float` —
only the first `size` elements are considered.
