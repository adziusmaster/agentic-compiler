## Reverse words

Write `reverse_words(text: Str) -> Str` that reverses the order of
words in a space-separated string while preserving single-space
separation between words.

- Input is guaranteed to have single-space separation (no leading,
  trailing, or repeated spaces).
- `"hello world"` → `"world hello"`.
- `""` → `""`.
- `"a"` → `"a"`.
